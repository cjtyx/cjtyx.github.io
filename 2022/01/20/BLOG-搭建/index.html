搭建BLOG

一.GitHub+hexo搭建

步骤：

1.注册一个GitHub账号

2.创建仓库

填写用户名.github.io 切记用户名必须和左边一致

3.启用GitHub pages

点击settings进入设置，往下拉找到“GitHub Pages”

4.下载git和node.js

安装好git后桌面上任意位置右键点击出现 git gui here和git bash here则说明好了

5.安装hexo，及其组件

自选一个地方创建文件夹hexo，在此文件夹下右击鼠标，选择“Git Bash here”

输入 npm install -g hexo进行安装

使用 hexo -v ,出现下面界面说明安装成功

然后hexo init 初始化,最后的“Start blogging with Hexo！”就说明成功了

输入 npm install,安装所需组件

输入 hexo g

输入hexo s，启动本地服务器，通过访问 http://localhost:4000，体验hexo ，不过应该访问127.0.0.1:4000 才能正确访问到页面

6.部署到GitHub

1.打开你所创建的文件，找到_config.yml文件,添加如下代码（用记事本或code打开）

repo：为仓库地址，branch：分支

这儿有个特别容易出错的点，就是branch 分支，教程上branch都为master，但是GitHub更新了一些，导致如果branch为master就部署不上去，应该为branch： main ，还有一点：冒号为英文冒号，且冒号后必须有空格

2.安装一个扩展

npm install hexo-deployer-git –save

3.输入cd ~/.ssh,如果出现入下图情况，说明不存在.ssh

解决方法：输入 $ ssh-keygen -t rsa -C “xxxxxxx@qq.com“(自己邮箱) 再次如下图进行操作，出现相同文件说明成功

4.输入eval “$(ssh-agent -s)”，添加密钥到ssh-agent

5.添加SSH Key到Github，点击“settings”

填写key

6.输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了（有一个会让你输入 yes）

7.设置你的账号信息

name和email后写自己的即可

8.部署到github hexo d 使用用户名.github.io 即可访问自己博客

9.传文件：将文件放到source/_posts 文件夹下，再部署就行

7.问题

_config.yml文件的配置：branch： main

hexo d 容易出现报错，可能是网络原因，可等等再试；可以输入git config –global http.sslVerify “false” 再试试

部署或者美化一般用 hexo clean hexo g (hexo s) hexo d

美化要在配置文件_config.yml修改，以及themes里主题下的_config.yml文件里修改

更换主题：1.下载主题文件放在themes文件下，修改配置文件下_config.yml的theme：改为主题文件的名字，再部署

二. 服务器+WordPress

步骤：

1.购买云服务器以及宝塔面板安装

云服务器安装centos，之前装过Ubuntu系统但是部署WordPress时出现了各种目前我无法解决的问题，重装centos系统后就格外顺利

用户名，密码可修改

使用ssh软件工具（Xshell）远程连接云服务器，使用公网IP，默认用户名root

输入宝塔面板安装命令

centos安装脚本：yum install -y wget && wget -O install.sh http://download.bt.cn/install/install.sh && sh install.sh

显示了宝塔后台地址和账号密码，均可在宝塔面板里修改，冒号后面为安全入口，也可修改

如果不能访问面板

在服务器中放行8888端口

然后访问宝塔面板，修改账号密码，绑定宝塔面板账号，完成实名认证，安装LNMP网站环境

2.下载WordPress安装包

下载地址：https://wordpress.org/

下载地址：https://zh-cn.wordpress.com/

3.在宝塔面板添加站点

没有买域名可以填写公网IP，创建FTP与数据库，用户名，密码自己填写

进入网站站点的根目录。后面安装任何网站程序的步骤都会在这个根目录下进行。只要新建一个站点，都会有唯一的站点根目录。站点根目录下的文件就是网站程序了。

上传刚刚下载的WordPress安装包，并解压，解压后生成了一个WordPress文件夹。将里面的文件剪切到上一层文件夹里就可以了

或者，由于站点根目录下安装包解压文件解压到”站点根目录/wordpress”目录下了。所以站点配置文件的root需要改为:

root /www/wwwroot/公网IP/wordpress;

4.在线安装WordPress

直接访问公网IP，开始安装WordPress

填写数据库名相关信息，也就是刚刚创建站点时填写的

提交后，就现在安装

填写网站站点相关信息

登录后就可访问网站后台管理页面

访问网站界面

之后可通过公网IP/wp-admin 访问网站后台管理页面，编辑博客

5.问题

就是服务器系统的选择，选得好就顺利，centos比较好

上传图片，打开hexo的配置文件_config.yml,找到 post_asset_folder，把这个选项从false改成true

需要安装插件 npm install hexo-asset-image –save 新建一篇文章hexo n “name”,在source/_posts文件下生成一个文件夹和一个md文件，将图片复制到文件夹里，在md上传图片，记得用相对路径
